## With icon

You can optionally add an icon to the left or to the right of your badge by using the `IconLeftComponent` and `RightIconComponent`.
In most cases, you will want to use only one side, but it's up to you if you want to add a different icon on each side.

<Canvas withSource="open">
	<Story
		name="With left icon"
		parameters={{
			docs: {
				source: {

						import { StarIcon, UserIcon } from "react-native-heroicons/outline"
						
						const App = () => {
							return (
								<>
									{/* With left icon */}
									<Badge IconLeftComponent={StarIcon}>Recommended</Badge>
						
					`
				}
			}
		}}>
		<Box width={375} alignItems="flex-start">
			<Badge IconLeftComponent={StarIcon}>Recommended</Badge>
		</Box>
	</Story>
	<Story
		name="With right icon"
		parameters={{

					code: dedent`
									{/* With right icon */}
									<Badge IconRightComponent={StarIcon}>Recommended</Badge>
						
					`
				}
			}
		}}>
		<Box width={375} alignItems="flex-start">
			<Badge IconRightComponent={StarIcon}>Recommended</Badge>

## Overriding the global color mode

You can optionally override the color mode for this specific component. You can choose between `light` and `dark`. 

By default, the component will use the globally active color mode.

<Canvas withSource="open">
	<Story
		name="Color mode override"
		parameters={{
			docs: {
				source: {
					code: dedent`
						import { Badge } from "@spirokit/core"
						import { StarIcon } from "react-native-heroicons/outline"
						
						const App = () => {
							return (		
								<Badge 
									size="sm" 
									IconLeftComponent={StarIcon} 
									colorMode="light">
									Recommended
								</Badge>
							)
						}
					`
				}
			}
		}}>
		<Box bg="white" p="4" alignSelf="flex-start">
			<Badge size="sm" IconLeftComponent={StarIcon} colorMode="light">Recommended</Badge>
		</Box>
	</Story>
</Canvas>

## Overriding the accent color

You can optionally override the accent color for this specific component. You can choose between `blue`, `orange`, `rose`, `amber`, `indigo`, `emerald`, `red`. 

By default, the component will use the globally active accent color, defined by the theme.
