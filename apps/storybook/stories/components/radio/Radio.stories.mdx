## Status flags

There are different flags that you can pass to each `Radio` component for further control over it:

`isDisabled`, `isHovered`, `isPressed`, `isFocused`, `isFocusVisible`, `isInvalid`

## Size

You can choose between `sm`, `md` and `lg` sizes for each `Radio` component. If you specified a size in the `Radio.Group`, it will be automatically propagated to each child, but you can optionally override that value on each child.

## Accent color override

You can optionally override the accent color for this specific component. You can choose between `blue`, `orange`, `rose`, `amber`, `indigo`, `emerald`, `red`. 

By default, the component will use the globally active accent color, defined by the theme.
