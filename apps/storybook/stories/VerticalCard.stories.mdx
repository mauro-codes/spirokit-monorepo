## Floating mode

By default, the `VerticalCard` component will use the content mode set to `floating`. This means that the image will take all the available space, leaving the content of the card floating on top of the image.
You can use this mode with and without an image, and the component will adapt accordingly.

## Fixed mode

You can also use the `VerticalCard` component with the content mode set to `fixed`. This means that the image will only take the top section of the card, leaving space for the content below.
You can use this mode with and without an image, and the component will adapt accordingly.

## Using badges

You can optionally display a `Badge` in the top right corner of the card. This is especially useful in the following scenarios:

-   Highlight featured or top news
-   Categories
-   Price badges for an e-commerce site

## Overriding the accent color

By default, the component will use the globally active accent color, defined by the theme.
