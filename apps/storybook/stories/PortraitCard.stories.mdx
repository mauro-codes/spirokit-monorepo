## Fixed mode

You can also use the `PortraitCard` component with the content mode set to `fixed`. This means that the image will only take the top section of the card, leaving space for the content below.
You can use this mode with an icon or with an image, and the component will adapt accordingly.

<Canvas withSource="open">
	<Story

		parameters={{
			docs: {

						import { Image, TitleThree, PortraitCard } from "@spirokit/core"
						import { NewspaperIcon } from "react-native-heroicons/outline"
						
						const App = () => {
							<>
								return (
									{/* Fixed mode with image */}
									<PortraitCard
										contentMode="fixed"
										TitleComponent={<TitleThree>Resting place of Australia's last convict ship</TitleThree>}
										AssetComponent={<Image source={{ uri: "https://i.imgur.com/1lSpdz3.png" }} alt="Image of a ship"></Image>}>
									</PortraitCard>
						
					`
				}
			}
		}}>
		<Box width={375}>
			<PortraitCard 
				TitleComponent={<TitleThree>Resting place of Australia's last convict ship</TitleThree>}
				AssetComponent={
					<Image
						source={{ uri: "https://i.imgur.com/1lSpdz3.png" }}
						alt="Image of a ship"></Image>
				}
				contentMode="fixed"
			>
			</PortraitCard>
		</Box>
	</Story>

			</PortraitCard>
		</Box>
	</Story>

## Using badges

You can optionally display a `Badge` in the top right corner of the card. This is especially useful in the following scenarios:

-   Highlight featured or top news
-   Categories
-   Price badges for an e-commerce site

## Overriding the global color mode

You can optionally override the color mode for this specific component. You can choose between `light` and `dark`. 

By default, the component will use the globally active color mode.

<Canvas withSource="open">
	<Story 
		name="Color mode override"
		parameters={{
			docs: {
				source: {
					code: dedent`
						import { Image, TitleThree, PortraitCard, Badge } from "@spirokit/core"
						
						const App = () => {
							return (
								<PortraitCard
									AssetComponent={
										<Image
											source={{ uri: "https://i.imgur.com/1lSpdz3.png" }}
											alt="Image of a ship"></Image>
									}
									TitleComponent={<TitleThree>Resting place of Australia's last convict ship</TitleThree>}
									BadgeComponent={<Badge>Travel</Badge>}
									colorMode="light">
								</PortraitCard>
							)
						}
					`
				}
			}
		}}>
		<Box width={375} p="4" bg="white">
			<PortraitCard 
				TitleComponent={<TitleThree>Resting place of Australia's last convict ship</TitleThree>}
				AssetComponent={
					<Image
						source={{ uri: "https://i.imgur.com/1lSpdz3.png" }}
						alt="Image of a ship"></Image>
				}
				BadgeComponent={<Badge>Travel</Badge>}
				colorMode="light">
			</PortraitCard>
		</Box>
	</Story>
</Canvas>

## Overriding the accent color

You can optionally override the accent color for this specific component. You can choose between `blue`, `orange`, `rose`, `amber`, `indigo`, `emerald`, `red`. 

By default, the component will use the globally active accent color, defined by the theme.

<Canvas withSource="open">
	<Story 
		name="Accent color override"
		parameters={{
			docs: {
				source: {
					code: dedent`
						import { TitleThree, PortraitCard, Badge } from "@spirokit/core"
						import { NewspaperIcon } from "react-native-heroicons/outline"
						
						const App = () => {
							return (
								<PortraitCard
									AssetComponent={NewspaperIcon}
									TitleComponent={<TitleThree>Resting place of Australia's last convict ship</TitleThree>}
									BadgeComponent={<Badge>Travel</Badge>}
									accentColor="rose">
								</PortraitCard>
							)
						}
					`
				}
			}
		}}>
		<Box width={375}>
			<PortraitCard 
				TitleComponent={<TitleThree>Resting place of Australia's last convict ship</TitleThree>}
					AssetComponent={NewspaperIcon	}
				BadgeComponent={<Badge>Travel</Badge>}
				accentColor="rose">
			</PortraitCard>
		</Box>
	</Story>
</Canvas>
